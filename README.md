
# HelloSDLAndroid
## このアプリについて
このアプリケーションは、SmartDeviceLink（以下、SDL)で動作するシンプルなAndroidのサンプルコードです。
SDLが提供している[Manticore](https://smartdevicelink.com/resources/manticore/)での利用を想定しています。
